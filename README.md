# How to develop?

```bash
# 安装依赖
npm i

# 启动
npm run dev

# 访问站点
http://localhost:3333
```
# 编码规范

- 使用 ``TypeScript`` 开发
- 方法，变量请设置好具体的数据类型
- 使用两个空格缩进（务必）
- 使用具有准确意义的变量/函数名称
- 建议开发工具：Visual Studio Code
